Técnicas de CSS

  1. In-line

    2. Técnica utilizada para alterar o estilo dentro da própria TAG HTML do próprio HTML.

  2. Interno

    3. Técnica utilizada para alterar o estilo geral das Tags, mas dentro da Tag HEAD no próprio HTML.

  3. Externo

    4. Técnica de CSS externo, ou seja, em outro arquivo CSS, que é o mais recomendado para deixar as configurações globais mais organizadas.
